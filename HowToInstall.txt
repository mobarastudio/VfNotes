You need to install this packages: make qt5-default libqt5webkit5-dev gstreamer1.0-plugins-base gstreamer1.0-tools gstreamer1.0-x

If you don't have, simply install by this commands:

sudo apt-get update
sudo apt-get install make g++ qt5-default libqt5webkit5-dev gstreamer1.0-plugins-base gstreamer1.0-tools gstreamer1.0-x

Naxt, open terminal in this folder and compile VfNotes:
make
./VfNotes
